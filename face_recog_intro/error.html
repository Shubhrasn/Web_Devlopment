Error During Installing Dlib

If you have following error then foolow the following steps:

Normally, all libraries are easilly installed but some libraries are not installed due to technical issu.

  • STEP-1: Download the CMake installer and install it: https://cmake.org/download/
  • STEP-2: Add CMake executable path to the Enviroment Variables: set PATH="%PATH%;C:\Program Files\CMake\bin" note: The path of the executable could be different from C:\Program Files\CMake\bin, just set the PATH accordingly. note: The path will be set temporarily, to make the change permanent you have to set it in the “Advanced system settings” → “Environment Variables” tab.
  • STEP-3: Restart The Cmd or PowerShell window for changes to take effect.
  • STEP-4: Download the Dlib source(.tar.gz) from the Python Package Index : https://pypi.org/project/dlib/#files extract it and enter into the folder.
  • STEP-5: Check the Python version: python -V. This is my output: Python 3.7.2 so I'm installing it for Python3.x and not for Python2.x note: You can install it for both Python 2 and Python 3, if you have set different variables for different binaries i.e: python2 -V, python3 -V
  • STEP-6: Run the installation: python setup.py install
